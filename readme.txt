
some git commands :

# initialise git
git init 

#basic command 
git add .
git commit -m "my msg"

# Looking for all other branches
git branch         				 

#look at a specific branch 
git checkout -b <my branch name>		 


#add a link to a remote repository with alias origin
git remote add origin https://github.com/<repository>  

git push -u origin master


#push a my branch to the origin
git push origin my-new-branch



git pull origin master
git log
